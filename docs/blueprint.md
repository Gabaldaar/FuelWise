# **App Name**: FuelWise

## Core Features:

- Registro de Combustible: Registra las compras de combustible con fecha, lectura del odómetro, tipo de combustible, precio/litro, costo total, litros y gasolinera.
- Ingreso de Datos de Combustible en Tiempo Real: Permite ingresar los datos de compra de combustible en tiempo real para garantizar un seguimiento actualizado. Calcula dinámicamente cualquiera de los valores de Precio/Litro, Costo Total o Litros, basándose en el ingreso de los otros dos.
- Análisis de Consumo de Combustible: Genera informes y gráficos con información como Km/litro promedio, gastos, promedios diarios y combustible usado. Permite configurar si se muestra Km/L o L/100Km.
- Recordatorios de Servicio: Envía notificaciones para el próximo mantenimiento, cambios de aceite y otros servicios según las fechas o el kilometraje.
- Estimación del Nivel de Combustible: Estima la próxima parada de reabastecimiento de combustible basándose en el consumo promedio de combustible, mostrado en kilometraje y fecha estimada. Esta función se implementa utilizando el LLM como una herramienta.
- Gestión de Vehículos: Gestiona múltiples vehículos con detalles como marca, matrícula, modelo, año y capacidad del depósito.
- Acceso Multi-Usuario: Permite que múltiples usuarios en diferentes dispositivos gestionen los mismos vehículos y sus datos.
- Optimización para Android: La aplicación está optimizada para su uso en dispositivos móviles Android.

## Style Guidelines:

- Color primario: Verde oscuro bosque (#228B22), inspirado en entornos naturales, refleja el viaje y la eficiencia del vehículo.
- Color de fondo: Verde muy claro y desaturado (#F0FAF0) para complementar el color primario en un esquema de luz.
- Color de acento: Un oro desaturado (#BDB76B) se utiliza para resaltar información importante.
- Emparejamiento de fuentes: 'Belleza' (sans-serif) para los encabezados, 'Alegreya' (serif) para el texto del cuerpo para crear una interfaz de usuario limpia y legible.
- Utiliza iconos claros y concisos para representar diferentes tipos de combustible, tipos de servicio y acciones del vehículo.
- Diseña un diseño de panel intuitivo para mostrar la información más relevante de un vistazo, con fácil acceso a los registros detallados.
- Implementa animaciones sutiles al actualizar datos o recibir notificaciones.
